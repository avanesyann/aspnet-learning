# :moneybag: Expense Tracker (ASP.NET MVC)

A simple and clean Expense Tracker web application built with **ASP.NET MVC**. This project helps users manage their daily income and expenses, track spending habits, and view summaries.

- Expense Categories Management
- Monthly Expense Reports
